# Robotron
**To run you first need to install the requirements:**

1. Install Python.
2. Run this in terminal:
```
git clone https://github.com/Strela196/robotron.git
```
3. go to robotron folder
4. run 
```
pip install -r requirements.txt
```
5. then run 
```
python robotron.py
```
